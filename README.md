## Topic Scheduling App

### How to install
```sh
git clone git://github.com/dijs/topic-scheduling-app
cd topic-scheduling-app
npm install
```

### How to configure
```sh
export FIREBASE_URL=<a firebase url>
export PORT=<port>
```
or create a `.env` with those variables

### How to build and run
```sh
npm run build
npm start
```
